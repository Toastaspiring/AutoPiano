起動方法
  Windows
    MIDIPlayer.batをダブルクリックで起動します。
    Windows10をお使いの方は、jarでもbatでもなく、JavaにWinMMパッチを当てることをおすすめします。
  他のOS
    ターミナルから起動します。
    $ java -jar MIDIPlayer.jar

Getting Started
  Windows
    Double-click the file MIDIPlayer.bat to start.
    If you're using Windows 10, you should apply WinMM patch to java, neither jar file nor batch file.
  Other OSes
    Run command in terminal.
    $ java -jar MIDIPlayer.jar
